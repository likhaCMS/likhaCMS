## LikhaCMS
LikhaCMS is a Web App Builder with Built-in UI Building Blocks,  
Drag and Drop Page Builder, Component Code Editor, Dev and Staging Environment

 - [x] Dev, Staging and Production Env
 - [x] Version Control

 - [x] Code Diff/Compare
 - [x] Global Reactive Variables

 - [ ] SEO (createMetaMixin)
